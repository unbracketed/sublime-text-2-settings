# Install

## Linux

```
cd ~/.config/sublime-text-2/Packages/User/
rm Preferences.sublime-settings
ln -s ~/code/sublime-text-2-settings/Preferences.sublime-settings .
rm Package\ Control.sublime-settings
ln -s ~/code/sublime-text-2-settings/Package\ Control.sublime-settings .
ln -s ~/code/sublime-text-2-settings/*.sublime-snippet .
```
